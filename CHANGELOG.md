## [Unreleased]

### Added
- 初始化项目结构
- 添加 `expert` 模板 - 以领域顶尖专家视角思考问题
- 添加 `clarify` 模板 - 通过追问深度理解需求后再给方案
- 添加 `debate` 模板 - 让AI扮演反方学者挑战你的观点
- 添加 `premortem` 模板 - 假设项目失败，提前发现风险
- 添加 `reverse` 模板 - 从成品范例逆向推导提示词
- 添加 `explain` 模板 - 同时获得通俗和专业两个版本的解释
