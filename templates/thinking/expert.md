# 用途

以领域顶尖专家的视角深度分析问题，帮助获得更专业、更有深度的思考。

# Prompt

我想探讨「{{领域}}」里的「{{问题类型/场景}}」。

先别回答。

请你先选一位最适合的领域顶尖名人专家来思考它。
可以是活人或历史人物，名字可以小众，但必须在该细分领域很专业。

如果你不确定该选谁，可以先反问我2个定位问题再选。

先输出：
1. 你选谁，他对应的细分领域
2. 为啥选他，三句话

然后再让我描述详细的问题。

# 使用示例

```
/expert LLM prompt管理
/expert 心理学 拖延症
/expert "产品设计" "用户留存"
```
