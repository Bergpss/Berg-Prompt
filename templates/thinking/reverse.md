# 用途

从优秀的成品范例中逆向推导出可复用的提示词，实现风格的稳定复刻。

# Prompt

这是我想要的成品范例：

{{成品范例}}

请你倒推一个提示词，让我用它能稳定生成同风格的内容。

并说明这个提示词里每一句的作用。

# 使用示例

```
/reverse [粘贴一篇你喜欢的文章]
/reverse [粘贴一段优秀的营销文案]
/reverse [粘贴一个出色的代码注释风格]
```
