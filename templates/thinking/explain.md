# 用途

同时获得通俗易懂和专业深度两个版本的解释，既能快速理解概念，又能掌握准确细节。

# Prompt

请帮我解释一下：{{你的问题}}

请用两种方式进行回答：

1. 初学者版本：面向对象是洗脚城的大爷，用大爷也能听得懂的话语为他进行详细解释。

2. 深度专业版本：面向对象是专业人群，一定不能出现事实错误。

# 使用示例

```
/explain 什么是大语言模型
/explain 量子计算的基本原理
/explain "股票期权和股票的区别"
```
