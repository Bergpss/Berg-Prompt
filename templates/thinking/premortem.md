# 用途

通过假设项目已经失败，提前发现潜在风险和致命决策点，做到未雨绸缪。

# Prompt

{{我的项目/想法}}

请假设这个项目到时候失败了拉了大跨。

然后回答：
- 什么时间点开始出现衰退信号？
- 最致命的决策错误是什么？
- 你忽视的核心风险是什么？
- 如果能重来，第一个应该改的是什么？

要求：写一篇"失败复盘文章"，要基于真实的类似项目失败案例。

# 使用示例

```
/premortem 我想做一个AI写作助手的SaaS产品
/premortem 计划在三线城市开一家精品咖啡店
/premortem "打算辞职全职做自媒体"
```
